Simple Java

Compiling and running java programs

Task 1 [1 p.]

Prepare a simplest java application using notepad. Save it in the file named Program1.java.

Task 2 [1 p.]

Compile and run the application created as the Task 1. Use command-line interface.

Task 3 [1 p.]

Add instruction for displaying the message "Hello world" to your program.

Task 4 [2 p.]

Prepare an application which declares and initializes variables of the following types:
  • boolean,
  • integer numbers,
  • real numbers,
  • characters.

Task 5 [1 p.]

Initialize the above variables with the least and the greatest available values of the corresponding types.

Task 6 [1 p.]

What is the (type of) the result of adding
  • a variable of type char to a variable of type int,
  • a variable of type int to a variable of type char,
  • a variable of type float to a variable of type double,
  • a variable of type byte to a variable of type int.

Task 7 [2 p.]

Prepare a program which demonstrates the problem of visibility of variables in the method main(). Comment out invalid references to variables.
